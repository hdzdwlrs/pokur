# pokur
Urbit Texas Hold'em app

Now available for distribution from ~dister-walrus-hodzod-walrus!

On an Urbit ship, search for ~dister-walrus-hodzod-walrus and you'll see the option to install %pokur and/or %pokur-server. For more information, join the group at ~bacrys/pokur.

This program was designed by ~hodzod-walrus as a grant project for the Urbit Foundation.
